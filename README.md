# 📚 Gestor de Tareas Académicas en Python

Este es un proyecto de consola desarrollado en Python diseñado para gestionar materias, exámenes y tareas pendientes de la carrera. Nació como una herramienta práctica para organizar mi propia cursada en el Técnico Superior en Análisis Funcional de Sistemas Informáticos.

---

### 🛠️ Desarrollo y Enfoque Técnico

El proyecto fue desarrollado utilizando **desarrollo asistido por Inteligencia Artificial (AI-assisted development)**. Mi rol en el proyecto se enfocó en:
- **Diseño Lógico:** Estructurar el flujo del programa y definir los requisitos del sistema (altas, bajas y consultas en memoria).
- **Control de Arquitectura:** Supervisar que el código se mantenga modularizado en funciones claras y legibles.
- **Pruebas y Debugging:** Ejecutar el script en entorno local (VS Code), analizar la persistencia de datos en memoria y validar el menú interactivo.

---

### ⚙️ Características

- **Visualización:** Listado claro de todas las tareas y exámenes almacenados temporalmente.
- **Interactividad:** Menú en consola para agregar nuevas asignaciones especificando materia, descripción y fecha límite.
- **Control:** Opción para marcar tareas como completadas.

---

2. Abrir una terminal en la carpeta del archivo.
